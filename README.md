# Sistema de Controle de Estoque – Mini-mundo (SQL)

Projeto acadêmico para prática de modelagem de banco de dados e comandos SQL (DML).

## Tecnologias

- Banco de dados: MySQL / PostgreSQL (ajuste conforme seu ambiente)
- Ferramenta: MySQL Workbench ou HeidiSQL

## Estrutura do Repositório

- `01_ddl_create_tables.sql` – Script opcional de criação das tabelas.
- `02_insert_dados.sql` – Inserts para povoar as tabelas principais.
- `03_select_consultas.sql` – Consultas (SELECT) com WHERE, ORDER BY, LIMIT e JOIN.
- `04_update_delete.sql` – Comandos de UPDATE e DELETE com condições.

## Como executar

1. Crie o banco de dados e as tabelas executando o script:

    SOURCE 01_create_tables.sql;
    SOURCE 02_insert_dados.sql;
    SOURCE 03_Select_consultas.sql;
    SOURCE 04_update_delete.sql;
